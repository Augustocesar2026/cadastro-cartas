Esse é um projeto escolar feito em C, chamado Super Trunfo de Países.
O Augusto desenvolveu um sistema onde você cadastra cartas de cidades, igual ao jogo Super Trunfo, mas com países. Cada carta tem informações como população, área, PIB e pontos turísticos.
O jogo funciona assim:

Você cadastra as cidades digitando os dados
O sistema calcula automaticamente a densidade populacional e o PIB per capita
Depois você pode comparar duas cartas e o sistema diz qual venceu em cada categoria

Por exemplo, na densidade populacional vence quem tiver o menor valor, mas nas outras propriedades vence quem tiver o maior.
O código foi feito do zero por ele e agora está sendo enviado pro GitHub, que é uma plataforma onde programadores guardam e mostram seus projetos.
